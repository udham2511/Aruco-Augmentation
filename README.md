
# Aruco Augmentation

Overlay video frames over aruco marker.

## Run Locally

Clone the project

```bash
  git clone https://github.com/udham2511/Aruco-Augmentation.git
```

Go to the project directory

```bash
  cd Aruco-Augmentation
```

Install dependencies

```bash
  pip install opencv-contrib-python
```

Run the script

```bash
  python main.py
```
